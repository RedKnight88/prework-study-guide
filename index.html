HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Common body elements include h1-6, p, a, ul, ol, li, and section.
  • Common attributes include class, id, name, charset, src, alt, href.

CSS

  • Selectors signify which elements within the HTML are styled.
  • Declaration blocks apply properties with specified values to the selectors.
  • 'Display: block' assigns each element to start on a new line.
  • Periods designate class selectors.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • This HTML element (in the head section) is mildly important: link rel="stylesheet" href="./assets/style.css"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetch and merge any commits from the tracking remote branch
  • git push: transmit local branch commits to the remote repository branch
  • git add: add a file as it looks now to your next commit (stage)
  • git commit [-m]: commit your staged content as a new commit snapshot (with a message in quotations after -m flag)

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • It works like C, just with implicit type conversions and simpler structure for function initialization.
